Elementos de Texto em CSS

Id (Identifier) e Class (Classes)

O Id em um arquivo CSS tem como finalidade identificar separadamente uma tag que necessite de uma edição separada.

O Id pode ser identificado em um código CSS com o símbolo de Hashtag(#), enquanto que a classe pode ser identificada com o símbolo de ponto. Vale ressaltar que segundo a W3C, o Id só pode identificar um elemento/tag.

Se caso a sua necessidade seja criar um identificador múltiplo, ou seja, que seja modificável dependendo da tag/elemento, uma classe poderá lhe ajudar.

Pseudo-Classes (:)

Precisam estar ligadas à um elemento ou uma classe. É identificada pelo símbolo de dois pontos (:)

A pseudo-classe referencia ao estado do elemento:

  • Pressionado (active)
  • Vazio (empty)
  • Mouse Sobre o Elemento (hover)
